# AI-Powered HR Analytics Dashboard

An end-to-end HR Analytics Dashboard built using Streamlit, Plotly, Python, and Gemini AI.

This project enables HR teams and business leaders to analyze employee attrition trends, workforce demographics, risk indicators, and AI-generated recommendations through an interactive dashboard.

---

## Live Dashboard

[Launch Application](https://ai-powered-hr-analytics-dashboard-b4hjwzca3rgyxfq8eztktr.streamlit.app)

## Features

### Data Upload
- Upload HR datasets in CSV format
- Automatic dataset validation
- Dataset preview and quality checks

### KPI Dashboard
- Total Employees
- Attrition Rate
- Average Employee Age
- Average Monthly Income

### Interactive Filters
- Department
- Gender
- Job Role
- Marital Status
- Education Field

### Visual Analytics
- Attrition by Department
- Attrition Rate by Department
- Attrition by Gender
- Attrition by Overtime
- Attrition Rate by Overtime

### Analytics Engine
- Automated Findings
- Business Recommendations
- Risk Identification
- Executive Summary

### Gemini AI Integration
- AI Findings
- AI Recommendations
- AI Risk Assessment
- AI Executive Summary

### Reporting
- PDF Report Generation
- PDF Download

## How to Run

### Clone Repository

```bash
git clone <repository-url>
cd AI_Business_Insights_Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create Environment File

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run Application

```bash
streamlit run app.py
```
